# ブランチ命名ルール

ブランチ名にはプレフィックスと担当者名を付け、作業内容と担当者が分かるようにします。

形式：

```text
<プレフィックス>/<作業内容>/<担当者名>
```

## プレフィックス一覧

| プレフィックス     | 用途       |
| ----------- | -------- |
| `feature/`  | 新機能追加    |
| `fix/`      | バグ修正     |
| `docs/`     | ドキュメント修正 |
| `test/`     | テスト追加・修正 |
| `refactor/` | コード整理・改善 |

## 命名例

```text
feature/login/yamada
fix/calculation-error/sato
docs/setup-guide/tanaka
test/calculator-test/suzuki
refactor/user-class/yamada
```

ブランチ名を確認するだけで、作業内容と担当者を把握できるようにします。
